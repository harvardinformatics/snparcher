# Tutorials

Tutorials are **learning-oriented** guides that take you through a series of steps to complete a project. They help you get started and build understanding through doing.

!!! tip "Who is this for?"
    Tutorials are for newcomers who want to learn snpArcher by working through complete examples.

## What to Expect

Each tutorial will:

1. Start from a clear beginning point
2. Guide you through each step with explanations
3. Result in a working outcome you can build upon
4. Focus on learning, not just completing tasks
